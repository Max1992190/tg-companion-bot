У меня есть рабочий Telegram-бот на Python (aiogram).
Не переписывай проект целиком.

Нужно добавить возможность просмотра диалогов пользователей с ботом
для анализа качества ответов и UX.

--------------------------------------------------
ЦЕЛЬ
--------------------------------------------------

Сохранять диалоги пользователь ↔ бот
и дать администраторам возможность их просматривать.

--------------------------------------------------
ТРЕБОВАНИЯ К ЛОГАМ
--------------------------------------------------

1. Сохранять ТОЛЬКО:
   - текст пользователя
   - текст ответа бота

2. НЕ сохранять:
   - user_id
   - username
   - имя пользователя
   - фотографии
   - платежную информацию

3. Формат записи (строго такой):

USER: <текст пользователя>
BOT: <ответ бота>
-----

4. Хранить только ПОСЛЕДНИЕ 100 диалогов
   (старые автоматически удалять)

5. Логи должны сохраняться в отдельный файл,
   например:
   dialog_logs.txt
   или dialog/dialog_logs.txt

--------------------------------------------------
КОГДА СОХРАНЯТЬ ДИАЛОГ
--------------------------------------------------

- Сохранять ТОЛЬКО если:
  • бот отправил текстовый ответ пользователю
  • ответ был сгенерирован OpenAI или быстрым ответом

- НЕ сохранять:
  • системные сообщения
  • меню
  • уведомления об оплате
  • сообщения поддержки
  • фото

--------------------------------------------------
ДОСТУП АДМИНИСТРАТОРОВ
--------------------------------------------------

Добавить команду для администраторов:

/logs

Право использовать команду имеют:
- основной администратор
- дополнительный администратор (8454575259)

--------------------------------------------------
ПОВЕДЕНИЕ КОМАНДЫ /logs
--------------------------------------------------

1. При вызове /logs:
   - показать последние 10 диалогов
   - в читаемом текстовом виде

2. Если логов нет:
   - показать сообщение:
     "No dialogs logged yet."

3. Команда доступна ТОЛЬКО администраторам

--------------------------------------------------
ВАЖНО
--------------------------------------------------

- Логи нужны ТОЛЬКО для анализа
- Логи НЕ отправляются пользователям
- Логи НЕ влияют на работу бота
- Логику оплаты, лимитов, бонусов НЕ менять

Добавить систему логирования и команду просмотра логов аккуратно.
